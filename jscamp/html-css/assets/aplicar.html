Ingeniero de software Frontend

Tech solutions está buscando un ingeniero de software con experiencia en desarrollo web. El candidato ideal tendrá experiencia en HTML, CSS, JavaScript. Si eres una persona proactiva y responsable, ¿quieres trabajar en un ambiente innovador y dinámico? ¡Esta oportunidad es para ti!

Responsabilidades

Requisitos

  • Excelentes habilidades de comunicación y trabajo en equipo
